# 📄 AI Resume Screening System with Tracing

## 📌 Overview

This project implements an **AI-powered Resume Screening System** that evaluates candidates against a given job description using **LLM-based pipelines**. It leverages **LangChain** for modular pipeline design and **LangSmith** for tracing, debugging, and monitoring.

---

## 🎯 Objective

* Build an intelligent resume evaluation system
* Implement **skill extraction, matching, scoring, and explanation**
* Enable **traceable and explainable AI outputs**

---

## ⚙️ Tech Stack

* Python
* LangChain (LCEL)
* LangSmith (Tracing)
* OpenAI / HuggingFace
* Jupyter Notebook / VS Code

---

## 🧠 Problem Statement

**Input:** Resume + Job Description
**Process:** Extraction → Matching → Scoring → Explanation
**Output:** Candidate Fit Score (0–100) + Justification

---

## 🔄 Pipeline Architecture

```
Resume → Skill Extraction → Matching → Scoring → Explanation → Tracing
```

---

## 🚀 Features

* ✅ Skill, experience, and tool extraction
* ✅ Resume-to-job matching logic
* ✅ Automated scoring (0–100)
* ✅ Explainable AI reasoning
* ✅ Modular LangChain pipeline (LCEL + `.invoke()`)
* ✅ LangSmith tracing for debugging

---

## 📈 Learning Outcomes

* Build modular LLM pipelines
* Design evaluation & scoring systems
* Implement explainable AI
* Debug using LangSmith

---


## 🚀 Conclusion

This project demonstrates the transition from **basic prompting to production-level AI system design**, focusing on **modularity, explainability, and traceability** — key skills for modern **GenAI and LLM engineering roles**.
