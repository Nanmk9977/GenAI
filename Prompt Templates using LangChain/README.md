# ✨ Mini Prompt Engine using LangChain

A simple yet powerful project demonstrating how to build **dynamic and reusable prompt systems** using LangChain.

## 🚀 Overview
This project replaces hardcoded prompts with structured templates using `PromptTemplate` and `ChatPromptTemplate`. It focuses on creating scalable and modular prompt pipelines for real-world Generative AI applications.

## 🔑 Features
- Reusable Prompt Templates  
- Multi-input prompt handling  
- Prompt variations (Teaching, Interview, Storytelling)  
- Role-based chat prompting  
- Input validation for robustness  
- Clean and modular pipeline design  

## ⚙️ Tech Stack
- Python  
- LangChain  
- Google Colab  

## 🔄 Workflow
User Input → Validation → Prompt Template → Dynamic Prompt Generation → Output

💡 *This project highlights the shift from static prompt writing to building scalable and reusable prompt engineering systems for real-world AI applications.*
